LossyArray

public struct LossyArray<Element> : Decodable where Element : Decodable

Decodable container for array of Decodable elements. Doesn’t fail decoding if one or few elements failed to decode. Useful for API response backward compatibility.

Usage example:

let models = try decoder.decode(LossyArray<Model>.self,
                                from: jsonData).elements
